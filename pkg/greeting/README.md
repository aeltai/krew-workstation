# Greeting Plugin for Rancher Dashboard

This is a simple greeting plugin that adds a new page to the Rancher Dashboard with a welcome message and a live clock.

## Features
- Displays a welcome message
- Shows current time that updates every second
- Accessible via the side navigation menu

## Development
To develop the plugin:
1. Run `yarn install` in the root directory
2. Run `yarn dev` to start development mode
3. Access the plugin at `https://127.0.0.1:8005` after logging in to Rancher
